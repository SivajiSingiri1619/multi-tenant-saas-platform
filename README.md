# Multi-Tenant SaaS Platform

A full-stack multi-tenant SaaS application built using Node.js, PostgreSQL, React, and Docker.

## Features
- Multi-tenancy with tenant isolation
- Role-Based Access Control (Super Admin, Tenant Admin, User)
- Projects and Tasks management
- JWT authentication
- Dockerized setup with auto migrations and seed

docker-compose up -d
